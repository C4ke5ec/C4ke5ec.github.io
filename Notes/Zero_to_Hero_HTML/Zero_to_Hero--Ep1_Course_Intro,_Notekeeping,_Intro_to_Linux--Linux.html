Simple Scripting



& allows for threading



## Simple network scanner

$1 - takes in user input (e.x. ipsweep.sh 192.168.0)
$ip - uses the sequence 1-254 as an IP
& - at the end does threading. Meaning that it will scan all of the ip's at the same time all at once.
./ipsweep.sh
#!/bin/bash
for ip in `seq 1 254`; do
ping -c 1 $1.$ip | grep "64 bytes" | cut -d " " -f 4 | tr -d ":" &
done

#### Fixed up the script to do a bit of error catching
#!/bin/bash

if [ "$1" == "" ]
then
echo "You forgot an IP address!"
echo "Syntax: ./ipsweep.sh 192.168.0."
